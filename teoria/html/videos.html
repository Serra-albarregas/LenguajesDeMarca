Videos

Como manejar uno de los recursos multimedia más importantes de internet

Incluir un video

Para incluir un video en HTML, utilizamos la etiqueta <video>. Esta etiqueta nos permite añadir diferentes atributos para controlar la reproducción del video, como:

  • muted: Silencia el video.
  • loop: Hace que el video se repita continuamente.
  • autoplay: Hace que el video comience a reproducirse automáticamente cuando la página se carga. (Nota: Algunos navegadores requieren que el video esté silenciado si se utiliza autoplay). 
<video controls loop muted autoplay src="ruta-del-video.mp4" type="video/mp4"></video>

Video con imagen de portada

La etiqueta <video> también permite establecer una imagen de portada o "poster", que se muestra antes de que comience la reproducción del video. Para ello, utilizamos el atributo poster. 

<video controls loop muted src="ruta-del-video.mp4" poster="ruta-de-la-imagen.jpg" type="video/mp4"></video>

Modificar el tamaño del reproductor y añadir bordes

Podemos ajustar el tamaño del reproductor de video añadiendo los atributos width y height a la etiqueta <video>. También es posible personalizar el diseño del reproductor usando CSS para, por ejemplo, añadir bordes. 

<video width="600" height="400" controls loop muted src="ruta-del-video.mp4" type="video/mp4"></video>

Para añadir bordes y otros estilos al video, podemos utilizar las clases de CSS, como en este ejemplo: 

video {
        width: 50%;
        display: block;
        margin: auto;
        border: 2px solid rgba(13, 112, 204, 0.247);
        border-radius: 6px;
}
