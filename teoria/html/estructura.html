Estructura básica de un documento HTML

Cómo se organiza un documento HTML y cuáles son sus principales etiquetas de estructura.

La plantilla mínima de un documento HTML

Todo documento HTML debe seguir una estructura básica que indica al navegador cómo interpretar el contenido. La plantilla mínima es la siguiente:

<!doctype html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
  </head>
  <body>
    <h1>Encabezado principal</h1>
    <p>Aquí va el contenido de la página.</p>
  </body>
</html>

<!doctype html>

Indica al navegador que el documento sigue el estándar HTML5.

<!doctype html>

<html>

Elemento raíz que contiene todo el documento. El atributo lang indica el idioma.

<html lang="es">
  <!-- Aquí van <head> y <body> -->
</html>

<head>

Contiene los metadatos y enlaces a recursos externos.

<head>
  <meta charset="UTF-8">
  <title>Mi primera página</title>
  <link rel="stylesheet" href="styles.css">
</head>

<body>

Contiene el contenido visible de la página.

<body>
  <h1>Bienvenido</h1>
  <p>Este es el contenido visible de la página.</p>
</body>

Encabezados <h1> a <h6>

Organizan el contenido en títulos y subtítulos con jerarquía.

<h1>Manual de HTML</h1>
<h2>Capítulo 1: Introducción</h2>
<h3>Sección 1.1: Historia</h3>
<h4>Subsección 1.1.1: Primeros navegadores</h4>

<p> (Párrafos)

Define bloques de texto corrido.

<p>HTML es el lenguaje estándar para crear páginas web.</p>

<header>, <nav>, <main>

Etiquetas semánticas que estructuran la página en zonas.

<header>
  <h1>Blog de programación</h1>
  <nav>
    <a href="index.html">Inicio</a>
    <a href="articulos.html">Artículos</a>
  </nav>
</header>

<main>
  <section>
    <h2>Bienvenidos</h2>
    <p>Este es el contenido principal.</p>
  </section>
</main>

<section>, <article>, <aside>

Dividen el contenido en partes temáticas o complementarias.

<section>
  <h2>Capítulo 1</h2>
  <p>Introducción a HTML.</p>
</section>

<article>
  <h2>Cómo usar etiquetas semánticas</h2>
  <p>Ayudan a que los buscadores comprendan el contenido.</p>
</article>

<aside>
  <h3>Recursos</h3>
  <ul>
    <li><a href="#">Guía oficial</a></li>
    <li><a href="#">Curso CSS</a></li>
  </ul>
</aside>

<footer>

Contiene el pie de página, normalmente con derechos de autor o enlaces secundarios.

<footer>
  <p>&copy; 2025 Mi página web</p>
</footer>

Ejemplo completo

<!doctype html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo completo</title>
</head>
<body>
  <header>
    <h1>Mi sitio web</h1>
    <nav>
      <a href="#">Inicio</a>
      <a href="#">Servicios</a>
      <a href="#">Contacto</a>
    </nav>
  </header>

  <main>
    <section>
      <h2>Bienvenidos</h2>
      <p>Este es el contenido principal.</p>
    </section>

    <article>
      <h2>Última noticia</h2>
      <p>Hoy hemos aprendido la estructura de HTML.</p>
    </article>
  </main>

  <aside>
    <h3>Recursos recomendados</h3>
    <ul>
      <li><a href="#">Guía oficial de HTML</a></li>
      <li><a href="#">Curso gratuito de CSS</a></li>
    </ul>
  </aside>

  <footer>
    <p>&copy; 2025 - Todos los derechos reservados</p>
  </footer>
</body>
</html>
