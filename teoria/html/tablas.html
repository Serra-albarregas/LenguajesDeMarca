Tablas

Se utilizan para organizar información con un formato de filas y columnas

Crear una tabla

Para crear una tabla básica en HTML, se utiliza la etiqueta <table>, que define el contenedor de la tabla. Dentro de esta etiqueta, se utilizan filas definidas por la etiqueta <tr> (table row) y celdas dentro de esas filas, las cuales son creadas por las etiquetas <td> (table data). Cada fila puede contener múltiples celdas.

  • <table>: define la tabla.
  • <tr>: define una fila de la tabla.
  • <td>: define una celda dentro de una fila.

<table>
    <tr>
        <td>Fila 1 Col 1</td>
        <td>Fila 1 Col 2</td>
        <td>Fila 1 Col 3</td>
        <td>Fila 1 Col 4</td>

Tablas con bordes

Para agregar bordes a una tabla, se pueden utilizar estilos CSS. El borde puede ser definido con la propiedad border, que permite especificar el grosor, estilo y color del borde. Para que se vea en todo el contorno de la tabla y las celdas, se aplica tanto a la tabla como a los elementos <td>.

  • border: define los bordes alrededor de la tabla y las celdas.

table {
    border: 1px solid black;
}

Celdas con bordes

Para agregar bordes a las celdas de una tabla, se puede aplicar CSS directamente a las celdas usando la etiqueta <td> con una clase específica que contenga estilos de borde. Es importante notar que si no se colapsan los bordes, estos aparecerán como duplicados entre las celdas.

table, td {
    border: 1px solid black;
}

Tabla y celdas con bordes colapsados

Al colapsar los bordes de una tabla con CSS utilizando la propiedad border-collapse: collapse;, los bordes de las celdas adyacentes se combinan en uno solo. Esto evita que se dupliquen los bordes entre celdas.

  • border-collapse: define cómo se manejan los bordes adyacentes en una tabla.

table, td {
    border: 1px solid black;
    border-collapse: collapse;
}

Combinar celdas

Para combinar varias celdas dentro de una tabla, se pueden utilizar los atributos colspan y rowspan. El atributo colspan permite que una celda ocupe varias columnas, mientras que rowspan hace que una celda ocupe varias filas.

  • colspan: define cuántas columnas abarcará una celda.
  • rowspan: define cuántas filas abarcará una celda.

<table>
    <tr>
        <td colspan="2">Fila 1 Col 1 y 2</td>
        <td>Fila 1 Col 2</td>
        <td>Fila 1 Col 3</td>
    </tr>
    <tr>
        <td>Fila 2 Col 1</td>
        <td>Fila 2 Col 2</td>
        <td>Fila 2 Col 3</td>
        <td>Fila 2 Col 4</td>
    </tr>
    <tr>
        <td colspan="2" rowspan="2">Fila 3 y 4 Col 1 y 2</td>
        <td>Fila 3 Col 3</td>
        <td rowspan="2">Fila 3 y 4 Col 4</td>
    </tr>
    <tr>
        <td>Fila 4 Col 3</td>
    </tr>
</table>

Cambiar el tamaño de la tabla

El tamaño de las celdas de una tabla se ajusta automáticamente al contenido, pero puede ser modificado manualmente usando CSS. Las propiedades width y height permiten establecer un tamaño fijo o relativo para las celdas. Si el contenido es más grande que el tamaño establecido, se ajustará y creará saltos de línea.

  • width: define el ancho de las celdas o la tabla.
  • height: define la altura de las celdas.

table {
    width: 500px;
}

Cambiar el tamaño de las celdas

Explicación: Para ajustar el tamaño de una celda en particular, se pueden usar las propiedades CSS width y height directamente en las etiquetas <td> o aplicando clases CSS a las celdas. Esto afectará el tamaño de toda la columna, ya que las celdas de una columna comparten el mismo ancho.

  • td { width: }: define el ancho de una celda.
  • td { height: }: define la altura de una celda.

td {
    width: 120px;
}

Ajustar el tamaño a la pantalla

Explicación: Para hacer que la tabla se ajuste a la pantalla de forma automática, se pueden utilizar unidades relativas como porcentajes en las propiedades width de la tabla y celdas. Esto asegura que la tabla se escale según el tamaño de la ventana del navegador, proporcionando una experiencia más adaptable en pantallas de diferentes tamaños.

  • width: 100%: hace que la tabla ocupe todo el ancho disponible.
  • max-width: configura un tamaño máximo para la tabla
  • min-width: configura un tamaño mínimo para la tabla
  • margin:auto: centra la tabla
  • td {width:25%}: especifica que cada celda debe ocupar el 25% de la tabla

table {
    width: 90%;
    max-width: 700px;
    min-width: 600px;
    margin: auto;
}

td {
    width: 25%;
}

Celdas de cabecera

Las celdas de cabecera dentro de una tabla se definen con la etiqueta <th>, que representa la cabecera de una columna o fila. Estas celdas tienen un estilo predeterminado de texto en negrita y están alineadas al centro, aunque pueden ser estilizadas mediante CSS.

  • <th>: define una celda de cabecera.

<table>
    <tr>
        <th>Fila 1 Col 1</th>
        <th>Fila 1 Col 2</th>
        <th>Fila 1 Col 3</th>
        <th>Fila 1 Col 4</th>

Selectores para filas pares e impares

Explicación: Con CSS, es posible alternar los estilos de las filas pares e impares de una tabla usando selectores como :nth-child(even) para las filas pares y :nth-child(odd) para las filas impares. Esto mejora la legibilidad de los datos al ofrecer un contraste visual entre las filas.

  • tr:nth-child(even): aplica estilos a las filas pares.
  • tr:nth-child(odd): aplica estilos a las filas impares.

tr:nth-child(even){
    background-color: #85c1e9;
}

tr:nth-child(odd){
    background-color: #deedf8;
}

Estructura de una tabla compleja con cabecera y pie

Una tabla puede dividirse en tres secciones principales: cabecera (<thead>), cuerpo (<tbody>) y pie (<tfoot>). La cabecera contiene las etiquetas <th> para definir las columnas, el cuerpo contiene los datos y el pie puede incluir filas de totales o resumen. Esto mejora la semántica y la accesibilidad de la tabla.

  • <thead>: define la cabecera de la tabla.
  • <tbody>: define el cuerpo de la tabla.
  • <tfoot>: define el pie de la tabla.
