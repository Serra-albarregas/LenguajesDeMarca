Selector Descripción
[href] El elemento tiene un atributo de tipo href
[href="#inicio"] El elemento tiene un atributo de tipo href con el valor #inicio
[lang~="es"] El elemento tiene un atributo de tipo lang que contiene una lista de valores, de los cuales uno es "es"
[href*="ini"] El elemento tiene un atributo de tipo href y su valor contiene la cadena "ini"

Combinadores CSS

TODO

Combinador Símbolo Descripción
Combinador descendiente Espacio Selecciona elementos dentro de otros en cualquier nivel
Combinador hijo > Selecciona elementos hijo directos
Combinador hermano adyacente + Selecciona elementos contiguos a otros en el mismo nivel
Combinador hermano general ~ Selecciona elementos que siguen a otros en el mismo nivel
Combinador universal * Selecciona todos los elementos en cualquier nivel
  • Azul - Combinador desdendente
  • Subrayado - Combinador universal
  • Cursiva - Hijo directo
  • Verde - Hermano adyacente
  • Rosa - Hermano general
