Selectores

CSS proporciona varias formas de seleccionar elementos HTML, permitiendo aplicar estilos específicos a cada uno de ellos. Los selectores son patrones que indican a CSS qué elementos deben ser afectados, y pueden combinarse o agruparse para crear reglas más complejas y personalizadas. Los selectores permiten un diseño detallado, desde seleccionar elementos específicos hasta aplicar estilos a grupos de elementos.

Estructura general

La estructura básica de un selector CSS incluye el selector propiamente dicho, seguido de un bloque de declaraciones. Cada declaración está formada por una propiedad CSS y un valor correspondiente. Por ejemplo, para definir el color de fondo de todos los párrafos, se usa el selector "p" seguido de una declaración de propiedad "background-color" con el valor deseado. Este patrón se repite para cada regla de estilo que se quiera aplicar.

Selector mediante etiqueta

Un selector de etiqueta en CSS selecciona todos los elementos de un tipo específico, por ejemplo, todos los elementos <p> para darles el mismo estilo. Este tipo de selector es útil cuando se desea aplicar un estilo uniforme a todos los elementos de un tipo en el documento HTML.

Selector con atributos

El selector de atributo selecciona elementos basados en la presencia o valor de un atributo específico. Este tipo de selector permite un control detallado, permitiendo, por ejemplo, seleccionar todos los enlaces que apunten a una URL específica o todos los elementos de un idioma particular. Los selectores de atributo son especialmente útiles en sitios multilingües o al trabajar con valores de atributos personalizados.

Selector Descripción
[href] El elemento tiene un atributo de tipo href
[href="#inicio"] El elemento tiene un atributo de tipo href con el valor #inicio
[lang~="es"] El elemento tiene un atributo de tipo lang que contiene una lista de valores, de los cuales uno es "es"
[href*="ini"] El elemento tiene un atributo de tipo href y su valor contiene la cadena "ini"

Combinadores CSS

Los combinadores CSS permiten seleccionar elementos en función de su relación con otros elementos en el DOM. Los combinadores ayudan a aplicar estilos a elementos dentro de otros, elementos contiguos, o elementos hermanos, proporcionando un control detallado sobre la disposición de los estilos en el árbol de documentos HTML. A continuación se describen algunos de los principales combinadores CSS:

Combinador Símbolo Descripción
Combinador descendiente Espacio Selecciona elementos dentro de otros en cualquier nivel
Combinador hijo > Selecciona elementos hijo directos
Combinador hermano adyacente + Selecciona elementos contiguos a otros en el mismo nivel
Combinador hermano general ~ Selecciona elementos que siguen a otros en el mismo nivel
Combinador universal * Selecciona todos los elementos en cualquier nivel

/* Selector descendente: aplica a todos los <p> dentro de un <div> en cualquier nivel */
div p {
    background-color: rgb(148, 148, 248);
}

/* Selector de hijo directo: aplica solo a los <p> que son hijos directos de un <div> */
div > p {
    font-style: italic;
}

/* Selector adyacente: aplica al <p> que sigue inmediatamente después de un <h5> */
h5 + p {
    background-color: rgb(122, 212, 122);
}

/* Selector de hermanos generales: aplica a todos los <p> que son hermanos de un <h6> */
h6 ~ p {
    background-color: rgb(212, 122, 122);
}

/* Selector universal: aplica a todas las etiquetas contenidas dentro del <div> */
div * {
    text-decoration: underline dotted;
}

Agrupación de selectores

La agrupación de selectores permite aplicar el mismo conjunto de reglas de estilo a múltiples selectores, sin tener que duplicar las declaraciones de estilo. Esto optimiza el CSS y facilita el mantenimiento del código, ya que los cambios de estilo solo necesitan realizarse en un lugar. Se puede agrupar selectores simplemente separándolos por comas.

.clase1{
    color: white;
}
.clase2{
    color: white;
}
.clase1, .clase2 {
    color: white;
}

Combinadores lógicos

Los combinadores lógicos en CSS permiten seleccionar elementos basándose en características específicas, lo que resulta útil para crear reglas avanzadas. Algunos de los combinadores lógicos más usados incluyen:

Combinador Descripcción
:is() Agrupador, igual que separando por comas
:has() Selecciona elementos que tengan hijos con ciertas características
:not() Selecciona elementos que no cumplan ciertas características

:is()

El pseudo-clase :is() permite agrupar selectores de forma eficiente dentro de un mismo bloque, similar a como se agruparían usando comas, pero optimizado para minimizar el código repetido. Con :is(), se pueden seleccionar varios tipos de elementos que comparten un estilo en común, simplificando las reglas CSS.

.clase1, .clase2 {
    color: white;
}

En este caso se selecciona tanto la clase "clase1" como la "clase2"

:is(.clase1, .clase2){
    color: white;
}

Si hay que seleccionar un par de clases dentro de un <div>, puede hacerse de la siguiente manera, usando un combinador descendente en dos elementos separados por comas.

div .clase1, div .clase2{
    color: white;
}

El código siguiente no serviría para sustituir el anterior.

div .clase1, .clase2{
    color: white;
}

Con :is() si que podríamos sustituirlo.

div :is(.clase1, .clase2){
    color: white;
}

:has()

La pseudo-clase :has() selecciona elementos que contienen un hijo con una característica específica. Esto es especialmente útil para seleccionar elementos padres cuando se quiere aplicar un estilo en función de un elemento secundario específico.

<div><p>Un div con un párrafo</p></div>
<div><a href="#inicio">Un div con un enlace</a></div>
<div><a href="#inicio">Otro div con un enlace</a></div>
div:has(a) {
    background-color: rgb(122, 212, 122);
}

:not()

El pseudo-clase :not() permite seleccionar elementos que no cumplen con ciertas características. Esto resulta útil para aplicar estilos a todos los elementos que no tienen una clase o atributo específico, facilitando el diseño de elementos en general o excepciones en un grupo.

<p class="parrafo1">Un párrafo con la clase "parrafo1"</p>
<p class="parrafo2">Un párrafo con la clase "parrafo2"</p>
<p class="parrafo3">Un párrafo con la clase "parrafo3"</p>
p:not(.parrafo1) {
    background-color: rgb(122, 212, 122);
}

Encadenando :not()

Es posible encadenar el pseudo-clase :not() para excluir varios tipos de elementos al mismo tiempo. Esto permite aplicar un estilo a un grupo amplio de elementos, dejando fuera varios tipos específicos, lo cual resulta útil en diseños complejos donde ciertos elementos necesitan ser estilizados de forma diferente.

<p class="parrafo1">Un párrafo con la clase "parrafo1"</p>
<p class="parrafo2">Un párrafo con la clase "parrafo2"</p>
<p class="parrafo3">Un párrafo con la clase "parrafo3"</p>
p:not(.parrafo1):not(.parrafo3) {
    background-color: rgb(122, 212, 122);
}

Combinando combinadores

CSS permite combinar múltiples combinadores y pseudo-clases para lograr estilos detallados. Estos combinadores permiten seleccionar elementos específicos, basándose tanto en relaciones en el DOM como en la exclusión o inclusión de atributos y clases. Esta combinación avanzada de combinadores es esencial para proyectos de diseño con estructuras de contenido complejas.

<p class="parrafo1">Un párrafo con la clase "parrafo1"</p>
<p class="parrafo2">Un párrafo con la clase "parrafo2"</p>
<p class="parrafo3">Un párrafo con la clase "parrafo3"</p>
p:not(:is(.parrafo1,.parrafo3)){
    background-color: rgb(122, 212, 122);
}
