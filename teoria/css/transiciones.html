Transformaciones y transiciones

Las transformaciones y transiciones son herramientas esenciales en CSS para mejorar la experiencia del usuario mediante efectos visuales. Las transformaciones permiten modificar la apariencia de los elementos mediante cambios como rotaciones, escalados y traslaciones. Por otro lado, las transiciones añaden suavidad a esos cambios, creando efectos animados al alterar las propiedades de los elementos. Estas herramientas son clave para lograr interfaces más atractivas e interactivas.

Transformaciones

Las transformaciones en CSS se aplican utilizando la propiedad transform. Permiten alterar la apariencia y posición de un elemento en el espacio mediante operaciones como traslaciones, rotaciones, escalados o inclinaciones. Son muy útiles para crear efectos visuales avanzados y proporcionar dinamismo a los diseños.

Traslación

La traslación en CSS permite mover un elemento desde su posición original a lo largo de los ejes X e Y del plano, utilizando la función translate(). Para influir en ambos ejes simultáneamente, puedes proporcionar dos valores separados por una coma dentro de translate(x, y). Por ejemplo, el primer valor afecta al desplazamiento horizontal (eje X) y el segundo al desplazamiento vertical (eje Y). Si solo deseas influir en un eje específico, puedes utilizar las variantes translateX() o translateY(). La primera afecta únicamente al eje X, desplazando el elemento horizontalmente, mientras que la segunda modifica la posición en el eje Y, moviéndolo verticalmente.

.element {
    transform: translateX(120px) translateY(50px);
}

Rotación

La rotación modifica la orientación de un elemento, girándolo en torno a un punto fijo (por defecto, su centro). La función rotate() permite definir el ángulo de rotación en grados, positivos para rotar en sentido horario y negativos para sentido antihorario.

.element {
    transform: rotate(20deg);
}

Escalado

El escalado en CSS, a través de la función scale(), permite cambiar el tamaño de un elemento en el plano sin modificar su posición. Puedes aplicar el escalado de manera proporcional en ambos ejes, X e Y, especificando un único valor como scale(valor). Si necesitas escalarlos de manera independiente, puedes usar dos valores separados, como scale(valorX, valorY), donde el primer valor afecta al eje X y el segundo al eje Y. Cuando se utilizan números menores que 1, el elemento se reduce proporcionalmente. Por ejemplo, un valor de scale(0.5) reduce el tamaño del elemento a la mitad. En contraste, los valores mayores que 1 amplían el tamaño del elemento. Si introduces valores negativos, el elemento no solo se escala, sino que también se invierte en el eje correspondiente. Por ejemplo, un valor de scale(-1) invierte el elemento horizontalmente en el eje X, mientras que scale(1, -1) lo voltea verticalmente en el eje Y. Esto puede ser útil para crear efectos visuales interesantes, como espejos o reflejos.

.element {
    transform: scale(2, 0.5);
}

.mirror {

Transiciones

Las transiciones en CSS permiten animar los cambios en las propiedades de un elemento cuando este pasa de un estado a otro. Por ejemplo, al cambiar su estado de reposo a uno interactivo, como al colocar el cursor encima, hacer clic o enfocar un elemento. Estas animaciones añaden dinamismo y suavidad, evitando saltos bruscos en los cambios visuales. Con las transiciones, puedes controlar el tiempo, la propiedad a modificar, la velocidad del cambio y si habrá algún retraso antes de iniciar el efecto. Esto las hace ideales para mejorar la experiencia del usuario al proporcionar retroalimentación visual fluida.

Desencadenante de las transiciones

Las transiciones se activan mediante eventos del usuario o cambios de estado, como al pasar el cursor por encima de un elemento (:hover), hacer clic (:active) o enfocar un campo (:focus). Estos desencadenantes determinan cuándo se inicia la animación especificada en la transición.

.element{
    transition: 1s;
}

.item1:hover {
    background-color: var(--blue-color);
}

.item2:active {
    background-color: var(--blue-color);
}

.item3:focus {
    background-color: var(--blue-color);
}

Cuando se aplica la transición

Una transición se muestra en el momento en el que un elemento o un elemento en un estado concreto está presente en la página. Esto significa que aplicar las propiedades de las transiciones sobre un elemento significará que estará animado en cualquier cambio de estado. Sin embargo, al aplicar las propiedades de una transición sobre un elemento en un estado provocará que la animación solo ocurra cuando dicho estado esté activo.

.item1 {
    transition: 2s;
}

.item1:hover {
    background-color: var(--blue-color);
}

.item2:hover {
    background-color: var(--blue-color);
    transition: 2s;
}

.item3 {
    background-color: var(--blue-color);
}

.item3:not(:hover) {
    background-color: var(--dark-color);
    transition: 2s;
}

Propiedades de la transición

Las propiedades de las transiciones definen el comportamiento de las animaciones en CSS. Estas incluyen la duración, la propiedad afectada, el tipo de aceleración y el retardo inicial. Usando estas configuraciones, puedes controlar cómo y cuándo los elementos cambian su apariencia.

transition-duration

transition-duration define cuánto tiempo tarda en completarse la transición. Puedes especificarlo en segundos (s) o milisegundos (ms). Es crucial para ajustar la velocidad de la animación según el diseño deseado.

.item1 {
    transition-duration: 0.5s;
}

.item2 {
    transition-duration: 1s;
}

.item3 {
    transition-duration: 2s;
}

.element:hover {
    background-color: var(--blue-color);
}

transition-property

transition-property determina qué propiedades del elemento se verán afectadas por la transición. Puedes especificar propiedades individuales, como background-color, o usar el valor all para aplicar la transición a todas las propiedades animables.

.element:hover {
    background-color: var(--blue-color);
    color: var(--dark-color);
    font-size: 1.5em;
    box-shadow: 0px 0px 5px 5px var(--dark-color);
    transform: rotate(30deg) translateY(-50px) scale(1.2);
}

    transition-property: background-color;
    transition-duration: 1s;

.item3 {
    transition-property: font-size;

    transition-duration: 1s;

.item5 {
    transition-property: transform;

    transition-property: all;

transition-timing-function

La propiedad transition-timing-function define cómo se desarrolla la transición a lo largo del tiempo, controlando la velocidad de la animación en cada punto. Esto se logra mediante funciones de tiempo predefinidas o personalizadas. A continuación, se detallan las opciones más comunes:

  • linear: La transición ocurre a una velocidad constante de principio a fin. Es ideal para movimientos uniformes.
  • ease: Es la opción por defecto. Comienza lentamente, acelera en el centro y desacelera al final, ofreciendo un efecto más natural.
  • ease-in: La transición comienza lentamente y acelera progresivamente hasta el final. Es útil para crear una sensación de aceleración.
  • ease-out: Inicia rápidamente y desacelera hacia el final. Se usa para dar un efecto de suavidad al final de la animación.
  • ease-in-out: Combina ease-in y ease-out, comenzando lentamente, acelerando en el medio y desacelerando hacia el final.
div {
    transition: 2s;
}

.container:hover div {
    width: 100%;
}

.item1 {
    transition-timing-function: linear;
}

.item2 {
    transition-timing-function: ease;
}

.item3 {
    transition-timing-function: ease-in;
}

.item4 {
    transition-timing-function: ease-out;
}

.item5 {
    transition-timing-function: ease-in-out;

transition-delay

transition-delay introduce un retraso antes de que comience la transición. Se define en segundos o milisegundos, permitiendo un control adicional sobre el momento en que se inicia la animación tras el evento desencadenante.

.item1 {
    transition-delay: 0.5s;
}

.item2 {
    transition-delay: 1s;
}

.item3 {
    transition-delay: 2s;
}

.element {
    transition-duration: 1s;
}

    background-color: var(--blue-color);

transition

transition es una forma abreviada que combina todas las propiedades de transición en una sola declaración. Incluye la propiedad afectada, duración, función de tiempo y retardo, simplificando la escritura del código.

.element {
    transition: background-color 0.5s ease-in 0.2s;
}

.element:hover {
    background-color: var(--blue-color);
    transform: scale(1.2);
}
