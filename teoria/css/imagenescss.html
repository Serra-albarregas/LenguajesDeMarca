Fondos

Cuando nos referimos a objetos, nos referimos también a contenedores, secciones, o cualquier elemento HTML que puede llevar un fondo o una imagen de fondo. Usar CSS para aplicar fondos a estos elementos ayuda a crear un diseño visual más atractivo y permite el control preciso sobre cómo se muestran y repiten los fondos.

Imágenes de fondo

CSS permite aplicar imágenes de fondo a los elementos HTML usando la propiedad background-image. Una imagen de fondo puede cubrir un área específica o repetirse para llenar un contenedor. A continuación, se explican las propiedades de CSS que ayudan a configurar cómo se despliegan estas imágenes.

Propiedad background-image

La propiedad background-image permite asignar una imagen de fondo a un elemento. Esta imagen se coloca detrás de cualquier contenido que el elemento contenga y puede personalizarse en combinación con otras propiedades de fondo.

elemento {
    background-image: url(../recursos/paisaje.png);
}
elemento {
    background-image: linear-gradient(#023047, #ffb703);
}

Propiedad background-repeat

Define cómo se repite la imagen de fondo.

Valor Descripción
repeat Repite la imagen de fondo en ambas direcciones (horizontal y vertical) para cubrir el área completa.
repeat-x Repite la imagen solo en el eje horizontal (de izquierda a derecha).
repeat-y Repite la imagen solo en el eje vertical (de arriba hacia abajo).
space Repite la imagen y ajusta los espacios entre las repeticiones para cubrir el área de manera uniforme.
round Ajusta la imagen para cubrir el contenedor sin cortar ninguna repetición. La imagen se escala ligeramente si es necesario.
no-repeat No repite la imagen de fondo. La imagen se muestra solo una vez.

Propiedad background-attachment

Controla si la imagen de fondo se desplaza o permanece fija cuando el usuario hace scroll en la página.

elemento {
    background-image: url(../recursos/paisaje_chico.png);
    background-attachment: scroll;
}

elemento {
    background-image: url(../recursos/paisaje_chico.png);
    background-attachment: fixed;
}

Propiedad background-position

Establece la posición inicial de la imagen de fondo dentro del contenedor.

elemento {
    height: 400px;
    background-image: url(../recursos/paisaje_chico.png);
    background-position: 50% 0%;
}

Propiedad background-clip

Determina hasta dónde se extiende el fondo dentro del contenedor del elemento.

  • border-box: La imagen de fondo se extiende hasta el borde del contenedor.
  • padding-box: La imagen de fondo se extiende solo hasta el borde del padding.
  • content-box: La imagen de fondo se muestra solo en el área del contenido.

elemento {
    height: 200px;
    width: 200px;
    margin: 5px;
    padding: 30px;
    border: 10px solid #02304771;
    background-image: url(../recursos/paisaje_chico.png);
    background-clip: content-box;
}

Propiedad background-size

Define el tamaño de la imagen de fondo. Puede ser un tamaño absoluto (en píxeles), relativo (en porcentajes), o los valores cover y contain.

  • cover: La imagen cubre todo el contenedor, recortándose si es necesario.
  • contain: La imagen se ajusta para estar completamente contenida en el contenedor, sin recortes.

elemento {
    width: 800px;
    height: 200px;
    background-image: url(../recursos/paisaje_chico.png);
    background-size: 800px 200px;
}

Multiples fondos

CSS permite aplicar varias imágenes de fondo en un mismo elemento. Esto es útil cuando se desea superponer imágenes o combinar fondos con diferentes configuraciones. Cada fondo se separa con una coma. La primera imagen que se establece es la última en dibujarse. Es importante que las imágenes seleccionadas contengan transparencia para que las de capas inferiores puedan visualizarse también.

elemento {
    height: 200px;
    width: 400px;
    background-image:
        url(../recursos/gatete.png),
        url(../recursos/paisaje_chico.png);
}

Aplicar propiedades a los distintos fondos

Cada fondo en CSS puede configurarse de forma independiente. Es posible aplicar diferentes posiciones, tamaños, y repeticiones a cada capa de fondo. Para ello, se debe indicar los distintos valores para cada fondo separados por comas en el mismo orden en que estos aparecen en la propiedad background-image.

elemento {
    height: 200px;
    width: 400px;
    background-image:
        url(../recursos/gatete.png),
        url(../recursos/paisaje_chico.png);
    background-repeat: no-repeat;
    background-position: 50% 60%, 0% 0%;
    background-size: auto, 400px 200px;
}

Efecto de viñeta con gradientes

Un gradiente radial puede usarse para crear un efecto de viñeta alrededor de una imagen de fondo, dando una apariencia enmarcada o sombreada.

elemento {
    height: 200px;
    width: 400px;
    background-image:
        radial-gradient(transparent, rgb(42, 33, 11)),
        url(../recursos/css.png),
        url(../recursos/papel.jpg);
    background-repeat: no-repeat;
    background-position: 0%, 50% 50%, 0% 0%;
    background-size: 100%, 50%, 100%;
}
