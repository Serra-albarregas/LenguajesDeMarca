Display

La propiedad display es fundamental en CSS para determinar cómo se representará un elemento en la página y cómo interactuará con otros elementos. Define el modelo de caja y la disposición de los elementos. Algunos valores importantes son:

  • inline: el elemento se representa en línea con otros elementos, similar al comportamiento de un fragmento de texto. No respeta propiedades como width y height, y su tamaño se ajusta al contenido.
  • block: el elemento se comporta como un contenedor de bloque, ocupando todo el ancho disponible en su contenedor padre. Comienza en una nueva línea y respeta propiedades como width, height, margin y padding.
  • inline-block: combina características de los elementos inline y block. Se posiciona en línea con otros elementos, pero permite definir width y height, lo que brinda flexibilidad al diseño.
  • flex: convierte el elemento en un contenedor flexible que permite distribuir y alinear elementos hijos de forma eficiente en un eje principal. Es ideal para crear diseños complejos con facilidad.
  • inline-flex: funciona de forma similar a flex, pero el contenedor se comporta como un elemento en línea, manteniendo la alineación con otros elementos en la línea actual.

block Los elementos se colocan en vertical. Ocupa lo que ocupa su padre
inline-block Los elementos se colocan en horizontal. No ignora width y height.

Inline y block

La diferencia principal entre los elementos con display: inline y display: block radica en cómo ocupan espacio y afectan el flujo del documento. Un elemento inline se coloca en la misma línea que el contenido adyacente y solo ocupa el espacio necesario para su contenido, sin permitir propiedades de ancho o alto. En cambio, un elemento con display: block se extiende por toda la anchura de su contenedor, comenzando en una nueva línea y empujando el contenido siguiente hacia abajo, permitiendo ajustar sus propiedades de tamaño (ancho y alto).

Los elementos div y span tienen un comportamiento distinto por defecto debido a sus tipos de display. Un div es un elemento de tipo block, lo que significa que ocupa toda la anchura disponible en su contenedor y provoca un salto de línea antes y después de él. Es ideal para agrupar elementos en secciones estructurales. En contraste, un span es un elemento inline, que ocupa solo el espacio de su contenido sin interrumpir el flujo del texto circundante. Esta característica lo hace útil para aplicar estilos a partes específicas de un texto sin cambiar la estructura de la página.

Inline-block

El valor inline-block en CSS permite una combinación de características útiles de los valores inline y block. Un elemento con display: inline-block se comporta de manera similar a los elementos inline en el sentido de que se sitúa en la misma línea que otros elementos. Sin embargo, a diferencia de los elementos puramente inline, permite definir width y height, lo que ofrece un control más preciso sobre el tamaño de los elementos sin forzar un salto de línea.

<div class="inline">Inline, intentando modificar su tamaño</div>
<div class="inline-block">Inline-block, modificando su tamaño</div>
.inline {
    display: inline;
    background-color: var(--red-color);
    width: 500px;
}

.inline-block {
    display: inline-block;
    background-color: var(--blue-color);
    width: 500px;
}

Centrar elementos

Para centrar elementos es importante tener en cuenta el display del elemento que se desea centrar. Se necesita un display de tipo bloque para poder centrar los elementos. También será importante tener en cuenta si las propiedades se aplican sobre un contenedor o sobre el contenido.

Centrar horizontalmente

Para centrar elementos de tipo bloque horizontalmente, una técnica común es definir un ancho específico para el elemento y aplicar margin: auto. Esto distribuye el espacio restante de manera uniforme a ambos lados del elemento, logrando así su alineación centrada dentro de su contenedor. Este enfoque es simple y efectivo, especialmente en elementos que tienen un display: block y están contenidos en un contenedor de ancho mayor.

.centrado {
    background-color: var(--red-color);
    display: block;
    width: 60%;
    margin: auto;
}
Boque centrado horizontalmente

Centrando verticalmente

Al aplicar align-content: center al contenedor, los elementos hijos se alinean verticalmente en el medio.

.contenedor {
    height: 200px;
    align-content: center;
}

.centrado {
    background-color: var(--red-color);
    display: block;
    width: 60%;
    height: 50%;
}
Bloque centrado verticalmente

Centrando texto

El centrado de texto en CSS se logra con la propiedad text-align: center. Esta propiedad se aplica al contenedor que contiene el texto y alinea horizontalmente el contenido de texto y otros elementos en línea. Funciona bien en elementos como <p>, <div>, y otros contenedores de texto, y es una forma rápida y sencilla de centrar títulos, párrafos y otros elementos textuales. Si se desea aplicar un centrado vertical sobre el texto se podría utilizar la propiedad align-content: center, también sobre el contenedor.

.centrado {
    background-color: var(--red-color);
    text-align: center;
    align-content: center;
    width: 60%;
    height: 80px;
}

Multicolumnas

column-count: Esta propiedad define el número de columnas en las que se desea dividir el contenido de un elemento. Cuando se especifica un column-count, el navegador intenta ajustar el contenido distribuyéndolo equitativamente entre las columnas, adaptando el ancho de cada una en función del espacio disponible.

.multicolumnas {
    column-count: 3;

Flex

Las cajas flexibles o flex son una herramienta poderosa en CSS que permite diseñar y distribuir el espacio de los elementos de manera eficiente.

Flex e inline-flex

El valor display: flex convierte un contenedor en un contenedor flexible, mientras que inline-flex hace lo mismo pero mantiene el contenedor en línea con otros elementos en la misma línea. Esto permite un control total sobre la alineación, orden y distribución de los elementos hijos, facilitando la creación de diseños complejos y adaptativos sin necesidad de flotar elementos o usar técnicas más antiguas como inline-block.

.contenedor{
    background-color: var(--dark-color);
    padding: 5px;
    display: ;
}

Dirección de los ejes

La propiedad flex-direction determina la dirección en la que se distribuyen los elementos hijos de un contenedor flex. Los valores incluyen:

  • row: disposición horizontal de izquierda a derecha (por defecto).
  • row-reverse: disposición horizontal de derecha a izquierda.
  • column: disposición vertical de arriba a abajo.
  • column-reverse: disposición vertical de abajo a arriba.

Esta propiedad es esencial para cambiar la orientación de los elementos en diseños flexibles y adaptativos.

.contenedor{
    background-color: var(--dark-color);
    display: flex;
    flex-direction: ;
}

Flex multilinea

La propiedad flex-wrap controla si los elementos hijos de un contenedor flex se ajustan al ancho del contenedor o se envuelven en una nueva línea. Los valores posibles son:

  • nowrap: los elementos permanecen en una línea única y no se envuelven (comportamiento predeterminado).
  • wrap: los elementos se envuelven a una nueva línea cuando no caben en una sola línea.
  • wrap-reverse: similar a wrap, pero la envoltura ocurre en dirección inversa.

El uso de flex-wrap es útil para crear diseños que necesitan adaptarse y reorganizarse automáticamente en función del espacio disponible.

.item {
    height: 50px;
    width: 50px;
    ...
}

.contenedor{
    background-color: var(--dark-color);
    display: flex;
    width: 250px;
    flex-wrap: ;
}

Gaps

En contenedores flex, las propiedades column-gap y row-gap permiten controlar la distancia entre las columnas y filas de los elementos. Estas propiedades ayudan a distribuir el espacio de manera uniforme, proporcionando una apariencia más limpia y estructurada al contenido. Son una mejora sobre el uso de márgenes individuales, ya que permiten separar los elementos de forma más controlada y consistente.

.contenedor{
    background-color: var(--dark-color);
    display: flex;
    width: 250px;
    flex-wrap: wrap;
    column-gap: 20px;
    row-gap:10px;
}

Alineación de contenido con flex

La alineación de contenido en un contenedor flex permite controlar la distribución de los elementos a lo largo de los ejes principales y secundarios. Con la ayuda de propiedades específicas, es posible ajustar la disposición y el espacio entre los elementos, así como su posición dentro del contenedor. Las propiedades más importantes para la alineación con flex son:

align-items: Esta propiedad controla la alineación de los elementos a lo largo del eje transversal (perpendicular al eje principal). Los valores más comunes son:

  • start: Alinea los elementos al inicio del eje transversal.
  • end: Alinea los elementos al final del eje transversal.
  • center: Centra los elementos en el eje transversal.
  • stretch: Estira los elementos para que llenen el espacio disponible en el eje transversal.
  • baseline: Alinea los elementos en la línea base de su contenido.

justify-content: Esta propiedad alinea los elementos a lo largo del eje principal (horizontal por defecto). Permite distribuir el espacio entre elementos de diferentes maneras. Los valores más comunes incluyen:

  • start: Alinea los elementos al principio del contenedor.
  • end: Alinea los elementos al final del contenedor.
  • center: Centra los elementos en el contenedor.
  • space-between: Distribuye los elementos con igual espacio entre ellos; el primer y último elemento se colocan al borde del contenedor.
  • space-around: Distribuye los elementos con espacio igual a ambos lados de cada elemento.
  • space-evenly: Distribuye los elementos de manera uniforme, incluyendo el espacio antes del primer y después del último elemento.

align-self: Esta propiedad permite alinear un elemento individualmente dentro de un contenedor flex, sobrescribiendo el valor de align-items para ese elemento específico. Sus valores son similares a los de align-items, permitiendo ajustes individuales para elementos particulares en el eje transversal.

.contenedor {
    background-color: var(--dark-color);
    display: flex;
    height: 200px;
    align-items: ;
    justify-content: ;
}

.self {
    align-self: ;
}

Orden de los elementos con flex

La propiedad order en un contenedor flex define la secuencia de los elementos hijos. Por defecto, todos los elementos tienen un valor de order igual a 0, pero se pueden asignar valores numéricos para reorganizar visualmente los elementos sin modificar la estructura del HTML. Un elemento con un valor de order más bajo se posicionará antes que otros con valores más altos, permitiendo reordenar los elementos de forma sencilla y efectiva para lograr diseños más dinámicos y adaptativos.

<div class="contenedor">
    <div class="item item1">1</div>
    <div class="item item2">2</div>
    <div class="item item3">3</div>
    <div class="item item4">4</div>
</div>
.item1 {
    order: 2;
}

.item2 {
    order: 2;
}

.item3 {
    order: 0;
}

.item4 {
    order: 1;
}
