Bordes con css

CSS permite agregar bordes a los elementos HTML. Puedes controlar el color, grosor, estilo y radio del borde para modificar su apariencia.

Propiedades de los bordes

Las propiedades de los bordes en CSS permiten modificar el aspecto visual de los bordes de un elemento. Se pueden ajustar propiedades individuales como el color, grosor, y estilo, o utilizar la propiedad border para aplicar todas de una sola vez.

Propiedad Valor Descripción
border-color Color Configura el color del borde
border-width Tamaño Configura el grosor del borde
border-style Estilo Selecciona un posible estilo para el borde
border Tamaño Estilo Color Atajo para incluir todas las propiedades como una
border-radius Radio Configura el redondeo de los bordes

Multiples parámetros

En CSS, es posible configurar los bordes de un elemento usando múltiples valores en propiedades como border-width, border-style y border-color. Estos valores permiten un control preciso sobre cada lado del borde (superior, derecho, inferior, e izquierdo).

Propiedad Parámetros Descripción
border-width tamaño Aplica el mismo grosor a todos los bordes
tamaño tamaño Aplica un grosor a top/bottom y otro a left/right
tamaño tamaño tamaño Aplica un grosor a top, otro a left/right y otro al bottom
tamaño tamaño tamaño tamaño Aplica un grosor a top, right, bottom y left en ese orden 
.elemento {
    margin: 20px;
    padding: 10px;
    border-style: solid dotted dashed;
    border-color: #2c3e50 #8ca6bf;
    border-width: 1px 2px 3px 4px;
}

Estilos de bordes

Los estilos de los bordes determinan cómo se dibuja la línea que rodea al elemento. Aquí hay varios estilos disponibles que puedes aplicar mediante la propiedad border-style. Algunos de los más comunes son:

Estilo Descripción
hidden Oculto
dotted Línea de puntos
dashed Línea de rayas
solid Línea contínua
double Doble línea contínua
groove Biselado con luz desde arriba
ridge Biselado con luz desde abajo
inset Profundidad hacia adentro
outset Profundidad hacia afuera

Bordes redondeados

Con la propiedad border-radius, es posible redondear los bordes de un elemento. Puedes aplicar un valor absoluto (en píxeles) o un valor relativo (porcentaje) para personalizar el nivel de redondeo de los bordes.

.redondeo_absoluto {
    border-width: 3px;
    border-color: #8ca6bf;
    border-style: solid;
    border-radius: 30px;
}

.redondeo_relativo {
    border-width: 3px;
    border-color: #8ca6bf;
    border-style: solid;
    border-radius: 30%;
}

Bordes con gradiente de color

Los bordes en CSS pueden no solo tener colores sólidos, sino también aplicar gradientes para lograr efectos visuales más complejos. El uso de gradientes en los bordes se consigue mediante la propiedad border-image, que te permite definir un borde con una imagen o un gradiente.

  • border-image-source: Especifica el origen de la imagen del borde. En este caso, es un gradiente lineal que varía entre tres colores
  • border-image-slice: Indica cómo se divide la imagen del borde en partes. Al usar valores de 1 1 1 1, se asegura que el gradiente se extienda uniformemente por todo el borde.
  • border-image-width: Define el grosor del borde basado en la imagen.
 .elemento {
    margin: 20px;
    padding: 10px;
    border-image-source: linear-gradient(#2c3e50, #00897b, #1de9b6);
    border-image-slice: 1 1 1 1;
    border-image-width: 0.25rem;
}
