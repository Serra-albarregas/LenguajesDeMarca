Introducción a CSS

Enlazar un documento css

Para enlazar una hoja de estilo externa a tu documento HTML, utilizamos la etiqueta <link>. Esta etiqueta debe colocarse dentro del elemento <head> del documento HTML. La hoja de estilo define cómo se verá el contenido HTML, aplicando estilos como colores, fuentes, márgenes y más.

El atributo rel="stylesheet" indica que el archivo enlazado es una hoja de estilos, y href contiene la ruta al archivo CSS.

<head>
    <link rel="stylesheet" href="estilos.css">
</head>

Sintaxis en CSS

La sintaxis de CSS está basada en reglas. Cada regla CSS tiene tres partes principales:

  • Selector: Especifica el elemento HTML al que se aplicarán los estilos.
  • Propiedad: Define qué aspecto del elemento se quiere modificar, como el color o el margen.
  • Valor: Indica el valor que se aplicará a la propiedad.

Las propiedades y valores van dentro de llaves {}, y cada propiedad se separa de su valor con dos puntos :. Cada par propiedad-valor termina con un punto y coma ;. 

SELECTOR {
    PROPIEDAD: VALOR;
    PROPIEDAD: VALOR;
    PROPIEDAD: VALOR;
}
p { 
    color: blue; 
    font-size: 14px; 
    margin: 10px; 
}

Unidades CSS

En CSS, las unidades de medida pueden ser absolutas o relativas. Las unidades absolutas se utilizan cuando se necesita un tamaño fijo (como píxeles o puntos), mientras que las unidades relativas se adaptan al entorno en el que se visualiza el contenido, como el tamaño del viewport o el tamaño de fuente.

  • Unidades absolutas: Son valores fijos que no cambian según el entorno o el dispositivo. Ejemplo: px, pt.
  • Unidades relativas: Se basan en el contexto o el entorno. Ejemplo: %, rem, em, vw, vh.

Unidad Descripción
Absolutas
px Píxeles en la pantalla
pt Puntos (Equivale a 0.35mm aprox)
Relativas
% Porcentaje del padre
rem Relativo al font-size del navegador (16px generalmente)
em Relativo al font-size del elemento actual
vw Relativos a la anchura del viewport
vh Relativos a la altura del viewport

Modelo de cajas

El modelo de cajas es fundamental para comprender cómo se estructura y diseña una página web. Cada elemento HTML se representa como una "caja", que consta de las siguientes partes:

  • Content: El contenido real del elemento, como el texto o imágenes.
  • Padding: Espacio entre el contenido y el borde.
  • Border: El borde alrededor del contenido y el padding.
  • Margin: Espacio fuera del borde, separando el elemento de otros elementos.

En CSS, la propiedad box-sizing determina si el padding y el border se incluyen dentro del ancho y alto especificados del elemento (border-box), o si se agregan al tamaño total del elemento (content-box).

Propiedad Descripción
Border Límite de la caja
Margin Espacio entre el borde y su exterior
Padding Espacio entre el contenido y el borde
Content Contenido de la caja

Referencia a una zona de un elemento

Al posicionar elementos en CSS, podemos hacer referencia a diferentes partes del espacio disponible utilizando propiedades como top, right, bottom y left.

Estas propiedades definen la distancia del elemento respecto a sus bordes contenedores, y son clave en el posicionamiento absoluto, relativo y fijo.

Propiedad Descripción
Left El elemento a la izquierda
Right El elemento a la derecha
Top El elemento de arriba
Bottom El elemento de abajo

Dimensiones

Las dimensiones en CSS se establecen con las propiedades width (ancho) y height (alto). Estas pueden definirse en unidades absolutas (como px) o relativas (como %, vw, vh). Si no se especifica un tamaño, el navegador ajustará automáticamente las dimensiones según el contenido o el contenedor.

Desbordamiento

La propiedad overflow controla lo que ocurre cuando el contenido de un elemento se desborda de su caja. Sus valores más comunes son:

  • visible: El contenido desbordado será visible fuera de su contenedor (comportamiento por defecto).
  • hidden: El contenido que no cabe en el contenedor será ocultado.
  • scroll: Se agregarán barras de desplazamiento para que el usuario pueda ver el contenido completo.
  • auto: Las barras de desplazamiento aparecerán solo si el contenido desborda el contenedor. 

.visible {
    width: 100px;
    height: 100px;
    font-size: 1.2em;
    border: 3px solid #2c3e50;
    overflow: visible;
}

.oculto {
    width: 100px;
    height: 100px;
    font-size: 1.2em;
    border: 3px solid #2c3e50;
    overflow: hidden;
}

.scroll {
    width: 100px;
    height: 100px;
    font-size: 1.2em;
    border: 3px solid #2c3e50;
    overflow: scroll;
}
