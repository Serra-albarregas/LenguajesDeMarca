Pseudoclases

Estructura general

Las pseudoclases son una forma de seleccionar elementos en CSS basada en condiciones específicas del estado o la posición de los elementos. A diferencia de los selectores comunes, las pseudoclases no seleccionan un elemento en función de sus características estáticas, sino que consideran el contexto o las interacciones del usuario. Para definir una pseudoclase, se utiliza el símbolo de dos puntos (:) seguido del nombre de la pseudoclase.

Tipos de pseudoclases

Las pseudoclases se dividen en varios tipos según su función en el estilo de los elementos. Cada tipo abarca distintas condiciones o estados de los elementos en una página HTML, y permite aplicar estilos de manera dinámica y contextual. Los principales tipos son:

  • Interacción: se aplican en función de acciones realizadas por el usuario, como pasar el cursor sobre un elemento o seleccionarlo.
  • Ubicación: seleccionan elementos según su relación de estado con el documento o su vinculación a enlaces.
  • Estructura: seleccionan elementos en función de su posición dentro de la estructura HTML.
  • Formulario: se utilizan con elementos de formulario, dependiendo de su estado como habilitado, deshabilitado, válido o requerido.

Pseudoclases de interacción

Las pseudoclases de interacción permiten aplicar estilos en función de cómo el usuario interactúa con un elemento en la página. Estas pseudoclases se activan temporalmente mientras el usuario realiza acciones como pasar el cursor o hacer clic.

:hover

La pseudoclase :hover se activa cuando el usuario pasa el cursor sobre un elemento. Es comúnmente utilizada para resaltar enlaces o botones al pasar el cursor por encima, mejorando la experiencia visual y de navegación.

<a href="#inicio">Enlace con hover</a>
a {
    color: rgb(27, 43, 73);
}

a:hover {
    color: rgb(122, 134, 185);
}
Enlace con hover

:active

La pseudoclase :active se activa mientras el usuario está presionando el elemento, como cuando hace clic en un enlace o un botón. El estado de "active" solo dura mientras el usuario mantiene el clic presionado, permitiendo efectos visuales inmediatos.

<a href="#inicio">Enlace con active</a>
a {
    color: rgb(27, 43, 73);
}

a:active {
    color: rgb(122, 134, 185);
}
Enlace con active

:focus

La pseudoclase :focus se activa cuando un elemento, como un campo de formulario, recibe el foco de entrada, ya sea por clic o navegación mediante el teclado. Esta pseudoclase es útil para destacar campos de entrada activos, mejorando la accesibilidad y la facilidad de uso.

<form method="get">
    <input type="text">
    <input type="text">
    <input type="text">
</form>
input {
    border: 2px solid rgb(122, 134, 185);
}

input:focus {
    border: 2px solid rgb(125, 204, 136);
    outline: none;
}

Pseudoclases de ubicación

Las pseudoclases de ubicación están relacionadas con el estado de los enlaces en una página o con su relación con el URL actual. Se usan comúnmente para controlar el estilo de los enlaces en sus distintos estados, como cuando se ha visitado o está activo.

:any-link

La pseudoclase :any-link selecciona cualquier elemento que sea un enlace, independientemente de si ha sido visitado o no. Facilita la aplicación de estilos globales a todos los enlaces de la página.

<a href="#inicio">Enlace</a>
:any-link {
    background-color: rgb(122, 212, 122);
}
Enlace

:link y :visited

Las pseudoclases :link y :visited permiten estilizar enlaces según si el usuario los ha visitado previamente o no. La pseudoclase :link selecciona enlaces que aún no han sido visitados, mientras que :visited selecciona aquellos que el usuario ya ha visitado.

<a href="http://www.google.com">Enlace</a>
a:link {
    background-color: rgb(212, 122, 144);
}

a:visited {
    background-color: rgb(122, 212, 122);
}
Enlace

:target

La pseudoclase :target selecciona un elemento que coincide con el fragmento de URL actual (la parte después del símbolo #). Es útil para resaltar o manipular secciones específicas de la página que son el objetivo de un enlace ancla.

<a href="#seccion1"><p>Enlace a la sección 1</p></a>
<a href="#seccion2"><p>Enlace a la sección 2</p></a>
<h4 id="seccion1">Sección 1</h4>
<p>Esta es la sección 1</p>
<h4 id="seccion2">Sección 2</h4>
<p>Esta es la sección 2</p>
h4:target {
    background-color: rgb(122, 212, 122);
    color: #333;
    padding: 5px;
}

Pseudoclases de estructura

Las pseudoclases de estructura permiten aplicar estilos según la posición de un elemento en la estructura del documento HTML. Facilitan el estilo de elementos en función de su ubicación en la jerarquía, como el primer hijo o el último elemento de un tipo.

:root

La pseudoclase :root selecciona el elemento raíz del documento, que generalmente es el elemento <html>. Es útil para definir variables CSS globales que se aplicarán a toda la página.

:first-child y :last-child

Las pseudoclases :first-child y :last-child seleccionan el primer o el último hijo de un elemento padre. Estas pseudoclases son útiles para estilizar el primer o último elemento en una lista o en una estructura de elementos anidados.

<div class="contenedor">
    <div class="lista">
        <div class="elemento">Primera lista, primer elemento</div>
        <div class="elemento">Primera lista, segundo elemento</div>
    </div>
    <div class="lista">
        <div class="elemento">Segunda lista, primer elemento</div>
        <div class="elemento">Segunda lista, segundo elemento</div>
    </div>
</div>
.contenedor :first-child {
    background-color: rgb(122, 212, 122);
}

:nth-child(), :nth-last-child() y :nth-child(A of B)

La pseudoclase :nth-child() permite seleccionar elementos en función de su posición dentro de su contenedor padre. La posición se puede especificar mediante una fórmula matemática, an + b, donde:

  • a: es un multiplicador que define el intervalo de selección.
  • n: representa un contador de incremento (por cada elemento en el contenedor).
  • b: es un desplazamiento o punto de partida para la selección.
Ejemplos comunes de expresiones en :nth-child() son:
  • 2n: selecciona cada segundo elemento (pares: 2, 4, 6,...).
  • 2n+1: selecciona cada tercer elemento (impares: 1, 3, 5,...).
  • 3n+1: selecciona el primer elemento y luego cada tercer elemento (1, 4, 7,...).
  • odd y even: se pueden utilizar como palabras clave, siendo equivalentes a 2n+1 (impar) y 2n (par) respectivamente.

Además de estas selecciones en base a posición, CSS introduce el uso del patrón :nth-child(A of B), donde:

  • A: es la misma expresión matemática que define la frecuencia de selección.
  • B: es un selector que define el tipo de elementos que se seleccionarán.
La expresión A of B permite mayor especificidad en la selección de elementos. En lugar de aplicar la fórmula a todos los hijos de un contenedor, esta pseudoclase selecciona únicamente aquellos que coinciden con el selector B, limitando así el ámbito de la selección solo a elementos de un tipo o clase específica.

Ejemplos prácticos de :nth-child(A of B) incluyen:

  • :nth-child(2 of .itemB): selecciona cada segundo elemento con la clase .itemB en su contenedor.
  • :nth-child(3n of p): selecciona cada tercer párrafo (<p>) dentro del contenedor.

El uso de :nth-child(A of B) es especialmente útil cuando se desea aplicar estilos a elementos específicos en un conjunto de diferentes tipos, evitando que el estilo afecte a todos los hijos del contenedor.

.grid-item() {
    background-color: #4CAF50;
    color: white;
}

:first-of-type, :last-of-type, :nth-of-type() y :nth-last-of-type()

Las pseudoclases :first-of-type, :last-of-type, :nth-of-type(), y :nth-last-of-type() se utilizan para seleccionar elementos en función de su tipo dentro de un contenedor. A diferencia de :nth-child(), que selecciona elementos en función de su posición relativa independientemente de su tipo, estas pseudoclases permiten seleccionar elementos basándose en el tipo específico de cada uno. A continuación se explica cómo funciona cada una:

  • :first-of-type: selecciona el primer elemento de un tipo específico dentro de su contenedor. Si, por ejemplo, se usa en un contenedor con varios tipos de elementos (como <p> y <div>), solo el primer elemento del tipo especificado será seleccionado.
  • :last-of-type: selecciona el último elemento de un tipo específico en el contenedor. Funciona de manera similar a :first-of-type, pero aplica la selección al último elemento de ese tipo en lugar del primero.
  • :nth-of-type(n): selecciona el enésimo elemento de un tipo específico dentro de su contenedor, basado en una fórmula an + b o palabras clave como odd y even. Esta fórmula sigue el mismo formato que :nth-child, pero aplicada solo a elementos del tipo especificado. Por ejemplo:
    • :nth-of-type(2): selecciona el segundo elemento de un tipo específico.
    • :nth-of-type(2n): selecciona cada segundo elemento de un tipo específico (pares).
  • :nth-last-of-type(n): funciona como :nth-of-type(n), pero la cuenta empieza desde el final del conjunto en lugar del principio. Por ejemplo:
    • :nth-last-of-type(1): selecciona el último elemento de un tipo específico en el contenedor.
    • :nth-last-of-type(3n): selecciona cada tercer elemento desde el final.

Algunos ejemplos prácticos de estas pseudoclases son:

  • :first-of-type: para estilizar el primer párrafo (<p>) dentro de un contenedor con varios tipos de elementos.
  • :last-of-type: para dar formato al último elemento <div> en un contenedor.
  • :nth-of-type(odd): para seleccionar elementos de un tipo en posiciones impares.
  • :nth-last-of-type(2): para seleccionar el penúltimo elemento de un tipo específico.

Estas pseudoclases son útiles cuando quieres aplicar estilos a elementos específicos de un tipo dentro de un contenedor, sin afectar a otros tipos de elementos en el mismo nivel.

<div class="contenedor">
    <div class="lista">
        <div class="elemento">Primer grupo, primer div</div>
        <p class="elemento">Primer grupo, primer párrafo</p>
        <div class="elemento">Primer grupo, segundo div</div>
        <p class="elemento">Primer grupo, segundo párrafo</p>
    </div>
    <div class="lista">
        <div class="elemento">Segundo grupo, primer div</div>
        <p class="elemento">Segundo grupo, primer párrafo</p>
        <div class="elemento">Segundo grupo, segundo div</div>
        <p class="elemento">Segundo grupo, segundo párrafo</p>
    </div>
</div>
.contenedor>div:first-of-type {
    background-color: rgb(122, 212, 122);
}

.lista p:last-of-type {
    border: 3px solid rgb(17, 180, 17);
}

.lista div:nth-of-type(2) {
    font-style: italic;
}

:empty

La pseudoclase :empty selecciona elementos que no tienen ningún contenido, ya sea texto, imágenes u otros elementos hijos. Es útil para aplicar estilos a contenedores vacíos y controlar su visualización.

<div></div>
<div>Div con texto</div>
<div><!-- div con comentario --></div>
div:empty {
    background-color: rgb(122, 212, 122);
}
Div con texto

Pseudoclases de formularios

Las pseudoclases de formulario permiten aplicar estilos en función de los estados de los campos de entrada, como si están habilitados, deshabilitados o requieren validación. Estas pseudoclases mejoran la accesibilidad y la interacción del usuario en los formularios.

:enabled y :disabled

Las pseudoclases :enabled y :disabled se aplican a los elementos de formulario en función de su estado de activación. La pseudoclase :enabled selecciona elementos de formulario habilitados para la interacción, mientras que :disabled selecciona los que están deshabilitados y no admiten interacciones del usuario.

<form method="get">
    <label for="f1Nombre">Nombre</label>
    <input type="text" id="f1Nombre"><br>
    <label for="f1Apellido">Apellido</label>
    <input type="text" id="f1Apellido" disabled>
</form>
input:enabled {
    background-color: rgb(122, 212, 122);
}

input:disabled {
    background-color: rgb(235, 129, 129);
    cursor: not-allowed;
}

:required

La pseudoclase :required selecciona los campos de formulario que tienen el atributo required, indicando que deben ser llenados antes de enviar el formulario. Esta pseudoclase es útil para resaltar visualmente los campos que necesitan completarse.

<form method="get">
    <label for="f2Nombre">Nombre</label>
    <input type="text" id="f2Nombre" required><br>
    <label for="f2Apellido">Apellido</label>
    <input type="text" id="f2Apellido">
</form>
input:required {
    background-color: rgb(235, 129, 129);
    border: 2px solid rgb(156, 45, 45);
}

:valid, :invalid, user-valid y user-invalid

Las pseudoclases :valid y :invalid se utilizan para aplicar estilos según la validez de los datos ingresados en campos de formulario, como aquellos con requisitos de formato específico. Además, las pseudoclases :user-valid y :user-invalid funcionan de manera similar, pero se activan solo después de la interacción del usuario con el campo.

<form method="get">
    <label for="f3Nombre">valid/invalid</label>
    <input class="valid" type="text" id="f3Nombre" required pattern="[A-Za-z]+"><br>
    <label for="f3Apellido">user-valid/invalid</label>
    <input class="user-valid" type="text" id="f3Apellido" required pattern="[A-Za-z]+"><br>
    <input type="submit" name="Enviar">
</form>
.valid:valid {
    background-color: rgb(122, 212, 122);
    border: 2px solid rgb(49, 156, 45);
}

.valid:invalid {
    background-color: rgb(235, 129, 129);
    border: 2px solid rgb(156, 45, 45);
}

.user-valid:user-valid {
    background-color: rgb(122, 212, 122);
    border: 2px solid rgb(49, 156, 45);
}

.user-valid:user-invalid {
    background-color: rgb(235, 129, 129);
    border: 2px solid rgb(156, 45, 45);
}
